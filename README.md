Hay palabras que se ofrecen, y otras que se protegen. <br/>
"TypeLock" es una tipografía que nace para custodiar esas palabras que valen más de lo que dicen. Como un cofre que solo revela su interior a aquel que tenga la llave, su forma cambia entre lo abierto, lo cerrado y lo bloqueado. <br/>
El usuario se convierte en guardián de cada letra, cada palabra, cada pensamiento. <br/>

Puedes probar "TypeLock" [aquí](https://samuelquiros-design.github.io/typelock/) para ver más detalles. <br/>
